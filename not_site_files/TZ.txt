В файле main.py будут собираться все функции, которые отвечают за вывод страниц.

Содержание файла main.py:
	- login Функция отображающая страницу для входа в аккаунт.
	И отвечает за вход пользователя в свой аккаунт
	- register Функция регестрирущая пользователя на сайт и отображает
	страницу регистрации нового пользователя.
	- index Функция, отображающая главную страницу магазина
	- shop Функция, выводящая сам интернет магазин с каталогом
	и фильтрами.
	- contact Функция, которая выводит контактную информацию о магазине
	- shop_single Функция, которая выводит страницу товара
	и др. странциы, которые появяться на сайте в будующем

Файл session_cookies.py будет собирать необходимые куки файлы

Будет создана папка forms, в которой будут появляться формы для заполнения бд

Будет создана папка data, в которой будут храниться в базы данных, необходимые
для работы сайта.

Будет создана папка data, в которой будут создаваться модели

Будет создана папка templates, которая будет хранить всю вёрстку сайта
